# Changelog

All notable changes to BatchForge.ai will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — 2026-03-19

### Added
- AI pipeline: Planner → Scaffold → Coder with intelligent planning
- Multi-provider fallback: xAI → OpenAI → Anthropic with automatic retry
- 3 quality levels: Fast, Balanced, Max
- VFS Mode: multi-file apps via Service Worker virtual file system
- Page Mode: single-page websites via HTML/CSS/JS
- Live preview rendering as AI writes code
- Continuous conversation: iterate while AI streams
- 30-level undo/redo with VFS snapshot persistence
- Export to single-file HTML or multi-file ZIP
- Fork & remix for any published project
- Stripe integration: Free, Pro ($12/mo), Team ($29/seat/mo), Enterprise tiers
- CDN resilience layer with cache-first strategy
- CodeMirror 6 editor integration
